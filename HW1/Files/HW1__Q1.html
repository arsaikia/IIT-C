As we can see from df_rec_sorted.shape we have 38185 rows containing unique address records. To plot a histogram we need to plot frequency of occurance on the OY axis. However, as we have 38K records it is not possible to plot each of the datapoint individually. Instead it makes sense to split 38185 in equal divisions.

Let us find the factors of the number which will help us find a suitable bin size for the histogram
